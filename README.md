# django-API

Proyecto que presenta un CRUD completo en python utilizando django rest framework, con gestion de logs y manejo de excepciones. Dicho codigo esta ejecutandose en un entorno virtual creado que se necesita acctivar para que  funcione, dicho entorno virtual se actica en la raiz del proyecto ejecutnado el siguiente codigo:

> ./venv/Scripts/activate

Una vez activado el entorno virtual, escribiendo el siguiente codigo puede ver las dependencias instaladas:

> pip freeze

Para crear el modelo en la base de datos se escribe los siguientes codigos en el siguiente orden:

> python manage.py makemigrations

> python manage.py migrate


Para ejecutar el proyecto utilice:

> python manage.py runserver
